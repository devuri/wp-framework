http_component_kernel()

helpers.php : 126

Initializes the App Kernel with optional multi-tenant support.

http_component_kernel(string $app_path[, string $options_file = 'app' ]) : Kernel

Sets up the application kernel based on the provided application directory path. In multi-tenant configurations, it dynamically adjusts the environment based on the current HTTP host and tenant-specific settings. It ensures all required environment variables for the landlord (main tenant) are set and terminates execution with an error message if critical configurations are missing or if the tenant's domain is not recognized.

Parameters
$app_path : string

The base directory path of the application (e.g., DIR).

$options_file : string = 'app'

Optional. The configuration filename, defaults to 'app'.

Tags
throws
Exception

If there are issues loading environment variables or initializing the App.

throws
Exception

If required multi-tenant environment variables are missing or if the tenant's domain is not recognized.

Return values
